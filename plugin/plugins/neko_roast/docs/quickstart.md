# NEKO Live 快速开始

这份教程给第一次使用的主播看。照着做就行，不需要懂开发环境。

## 开播前

1. 打开打包版 N.E.K.O。
2. 进入“插件管理”。
3. 打开 “NEKO Live” 插件详情。
4. 切到“面板”。

## 连接直播间

1. 在“控制台”里选择直播平台。
2. 打开“账号”并登录：B站推荐扫码登录，抖音导入凭据。B站无账号连接仅作为需要手动启用的兼容性兜底。
3. 填直播间号，或直接粘贴直播间链接。
4. 确认显示的房间标题、主播名和直播状态是你的直播间。
5. 点击“开始监听”。
6. 看到监听状态变成“已连接”或“接收中”，就说明弹幕通道已经接上。

## 让 NEKO 开口

1. 先让直播间发一条测试弹幕，比如“猫猫在吗”。
2. 看 NEKO 是否回复。
3. 如果回复了，就可以正常直播。
4. 如果没回复，点“刷新状态”，再看面板顶部状态：
   - “不能开播”：先检查直播间号、是否开始监听、输出通道是否可用。
   - “已阻塞”或“未连接”：重新开始监听。
   - “冷却中”或“等待”：等几秒，再发一条明确的问题测试。

## 常用位置

- “控制台”：填直播间、查询房间、开始/停止监听、切换直播模式。
- “直播间互动”：开启或关闭弹幕锐评，调整锐评强度。
- “观众”：看最近弹幕处理结果和观众档案。
- “设置”：分为“安全与性能 / 数据与隐私 / 帮助与高级”。安全暂停建议保持开启；忙时处理用“谨慎 / 标准 / 宽松”三档，默认推荐“标准”。这里也可以查看本地档案位置、重新打开新手教程或启用开发者模式。

## 直播主题怎么填

直播主题写一句普通、具体、不会泄露活动信息的话。它会帮助 NEKO 记住本场直播在聊什么。

可以这样写：

- 今天轻松聊天，和观众聊游戏、日常和最近在玩的内容。
- 本场是杂谈直播，重点是接弹幕、陪观众聊天，避免聊展会和未公开安排。
- 今天主打轻松陪伴，观众问什么就自然接，不主动提保密内容。

不要写：

- 展览、项目组、未公开活动、内部安排。
- 太长的剧本。
- 需要 NEKO 扮演真人主播、工作人员或后台操作员的内容。

## 直播中如果出问题

- 弹幕很多但 NEKO 不回：先等几秒，NEKO 会优先挑更值得回的弹幕。
- NEKO 回复太频繁：到“控制台”的“节奏设置”里降低说话频率或加长冷却时间。
- NEKO 太安静：发一条明确问题，比如“猫猫今天想聊什么”。
- 房间接不上：停止监听，再重新查询直播间并开始监听。
- 不确定状态：先点“刷新状态”并查看“控制台”；需要链路解释时，在“设置 → 帮助与高级”开启开发者模式，再到开发者子界面查看运行结果。

## 下播后

1. 点击“停止监听”。
2. 确认监听状态不再是“接收中”。
3. 需要继续测试时，再重新查询直播间并开始监听。
